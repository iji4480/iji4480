# JUNSU JO

## My Introduce
저는 유한대학교 정보통신 영상보안 전공을 졸업하고 국비학원 6개월 과정을 밟으며 개발자의 길을 가고있는 취준생입니다.<br>
학교에서는 전공과목에 흥미를 느끼지 못했지만, 학원을 다니면서 배운 기술로 프로젝트를 진행해 오면서 개발에 흥미가 생겨서<br>
아이디어만 있다면 뭐든 구현해낼 수 있는 코딩 실력을 가져야 겠다는 생각이 들었습니다.
